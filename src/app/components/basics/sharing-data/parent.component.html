Sharing Data Between Parent and Child Components

1. Input and Output

First of all, the idea of Input and Output is to exchange data between components. They are a mechanism to send/receive data from one component to another. Input and Output give a child component a way to communicate with its parent component. Input lets a parent component update data in the child component. Conversely, Output lets the child send data to a parent component.
